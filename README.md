# Bushsong | Artisanal Australian Craft

Bushsong is a premium e-commerce platform showcasing handcrafted Australian furniture and items born from reclaimed timber. Every piece is unique, rescuing discarded shipping pallets and breathing new life into forgotten wood.

## ✨ Premium UI Revamp (v1.1.0)

We have recently overhauled the entire user experience to provide a "WOW" factor that matches our artisanal craftsmanship.

- **Vanilla CSS Architecture:** Migrated from Tailwind to scoped CSS Modules for maximum design flexibility.
- **Dynamic Animations:** Powered by `framer-motion`, featuring staggered entrances, scroll-reveal effects, and tactile hover interactions.
- **Modern Aesthetics:** Implementation of glassmorphism, rich color palettes (Charcoal Green & Gold), and refined typography.
- **Performance Optimized:** Clean, hardware-accelerated animations and optimized component structures.

## 🚀 Features

- **Unique 1-of-1 Inventory:** Every product is a one-off handcrafted item.
- **Real-time Reservations:** Dynamic cart system with 10-minute hold periods to ensure fair access to limited items.
- **Stripe Integration:** Secure checkout process.
- **Sustainability Focused:** 100% reclaimed materials and eco-friendly practices.

## 🛠️ Tech Stack

- **Framework:** [Next.js 14](https://nextjs.org) (App Router)
- **Styling:** Vanilla CSS Modules
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Database:** Prisma with SQLite
- **Icons:** Lucide React

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
